# Prototype Help Text

The TRACE Rehabilitation Engineering Research Center (RERC) has created many prototype devices to demonstrate accessible systems.
This repository contains help text from some of the prototypes.

Note that the help text underwent differing levels of review and testing.
They might not be optimal for today or for different types of systems, but may serve as a starting point.
